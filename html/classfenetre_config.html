Description détaillée

Classe abstraite permettant de sauvegarder le contexte de toutes les fenêtres inclues dans MainWindow.

Voir également
MainWindow, SousFenetre, fenetreElementaryRule, fenetreGameOfLife, fenetreFeuForet

Cette classe abstraite fourni un comportement à l'interface et permet de sauvegarder le contexte de toutes les fenêtres inclues dans MainWindow. Elle concerne la partie gauche de l'interface.
